## 什么是Aplayer?

:::tip 说明

由于这里示例音乐链接是网易云的,可能因为网络问题会无法播放

:::

## 安装APlayer

```shell
yarn add APlayer
```

## 注入组件

在你的博客首页对应的markdown文件的widgets中注入一个有id的div
```md
---
layout: blog
widgets:
  - name: "音乐播放器"
    html: "<div id='custom1'></div>"
---
```

## 加载播放器

APlayer的更多参数可以参考[APlayer官网](https://aplayer.js.org/)

```css

}
```
